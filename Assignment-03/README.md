# Pointer Playground

## Objective
To understand the basic concepts of pointers in C programming by performing operations such as displaying memory addresses, dereferencing pointers, swapping two numbers using pointers, traversing an array using pointers, and demonstrating pointer arithmetic.

## Problem Statement
Write a menu-driven C program to demonstrate pointer concepts. The program should:
- Display the memory address of a variable.
- Access the value of a variable using a pointer.
- Swap two numbers using pointers.
- Traverse an array using pointers.
- Demonstrate pointer arithmetic.

## Algorithm
1. Start the program.
2. Display the menu with different pointer operations.
3. Read the user's choice.
4. Perform the selected operation:
   - Display memory address.
   - Dereference a pointer.
   - Swap two numbers using pointers.
   - Traverse an array using pointers.
   - Demonstrate pointer arithmetic.
5. Repeat the menu until the user selects Exit.
6. End the program.

## Time Complexity

| Operation | Time Complexity |
|-----------|-----------------|
| Display Memory Address | O(1) |
| Dereference Pointer | O(1) |
| Swap Two Numbers | O(1) |
| Traverse Array | O(N) |
| Pointer Arithmetic | O(N) |

## Space Complexity

- **O(N)** (For storing the array used in traversal and pointer arithmetic.)

## Sample Input

```
Choice: 3

Enter two numbers:
10 20
```

## Sample Output

```
After Swapping:
a = 20
b = 10
```

## Learning Outcome

- Learned the basics of pointers in C.
- Understood memory addresses and pointer variables.
- Accessed values using dereferencing.
- Swapped values using pointers.
- Traversed arrays using pointers.
- Performed pointer arithmetic.
- Improved understanding of functions and arrays.
